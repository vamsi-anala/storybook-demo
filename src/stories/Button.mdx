# Button Component

The `Button` component allows you to define reusable UI elements with customizable properties such as `label`, `color`, and `isDisabled`.

## Properties

- **label**: Specifies the text displayed on the button, making it clear and identifiable to users.
- **color**: Defines the background color of the button for visual customization.
- **isDisabled**: A boolean value that disables the button and visually indicates its inactive state when set to `true`.
